# JEngine
A basic C++ game engine

## Tools used
| Tool | Use |
| ----------- | ----------- |
| GLFW/Glad | Graphics/OpenGL stuff |
| glm | Maths |
| Dear ImGui | UI |
| spdlog | Logging |
| stb_image | Image Loading |
| premake | Build System |
